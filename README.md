# Todo Command Line Application

## Installation
```bash
git clone https://github.com/duyanhitbe/todo-cli
```
Then:
```bash
cd todo-cli
```
And:
```bash
make build
```
You can you intermediately by:
```bash
./bin/todo-cli -h
```
If you want to use it in global
```bash
go install github.com/duyanhitbe/todo-cli@latest
```
